Numbers

Primitive number types are divided into two groups: Integer types and Float types.

Integer types stores whole numbers, positive or negative (such as 123 or -456), without decimals. They are:

  • byte
  • short
  • long
  • int

You can declare an integer type as follows: int x = -54;

Floating point types represents numbers with a fractional part, containing one or more decimals.They are:

  • float
  • double

You can declare an floating point type as follows: float x = 243.57;

Booleans

Boolean types are used for cases that only have two types. For example: yes or no, true or false, on or off.

The boolean data type in Java can have the value true or false. True is sometimes represented by 1 and false by 0.

You can declare a boolean type as follows: boolean iLoveCoding = true;

Characters

The char data type is used to store a single character. The character must be surrounded by single quotes, like 'A' or 'c':

ASCII values can also be used to define char variables, using a number code instead of a letter.

You can declare a char type as follows: char c = 'c';

Strings

The string data type is a collection of characters to form text.They use double quotation marks.

There are special methods that can be used on strings to access certain characters, combine strings, or even cut a string in half.

You can declare a string type as follows:String s = "Hello, World!"

OOP

In Java, you can create your own objects, like Strings, Lists, and Arrays, for your own purposes. We call this object oriented programming.This is the next step in Java, so make sure you are comfortable with all of the variables and other objects first!
